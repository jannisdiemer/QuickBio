# QuickBio

A modern personal profile platform that allows users to create and share a customizable digital identity.

## 🚀 Features
- Custom profile pages
- Social media integration
- Account management (create, edit, delete)
- Email verification & contact system
- Clean UI with responsive design

## 🛠 Tech Stack
- PHP
- MySQL
- HTML, CSS, JavaScript
- Bootstrap
